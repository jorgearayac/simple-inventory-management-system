# Simple Inventory Management System
Console-based inventory management system built in C# for learning purposes.

## Description
The application allows users to manage a list of products by adding, viewing, editing, deletingm adn searching for items in an inventory.

The main goal of this project is to practice C# fundamentals, object-oriented programming, and basic Git/GitHub workflows.

## Features
- Add a product (name, price, quantity) to the inventory
- View all products in the inventory
- Edit an existing product
- Delete a product
- Search for a product by name
- Exit the application
